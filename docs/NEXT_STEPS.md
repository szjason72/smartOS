## 🔧 技术债务和注意事项

### 需要注意的事项：

1. **数据库选择**
   - 当前使用 PostgreSQL
   - 集成到 zervi.test 时需要迁移到 MySQL
   - 建议：先完成功能，再迁移

2. **MQTT 集成**
   - 当前配置了 Mosquitto
   - 需要实现 MQTT 客户端
   - 建议：使用 Spring Integration MQTT

3. **真实设备通信**
   - 当前使用 Mock 设备
   - 后续需要实现真实 BC4 Pro 通信
   - 建议：保持 Mock 和真实设备的接口一致
